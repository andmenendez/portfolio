Solace

Mental health is apriori to all philosophical & esoteric principles. Solace is a project that I started with a friend to create a more conducive environment for individuals to find therapists or psychiatrists.

I created this platform from scratch via django & it is still in early development. I intend to put a lot more time and effort into developing this application into something that could possibly be used. This is a project I am passionate about and a project that is just technical enough to be challenging and yet do-able.
